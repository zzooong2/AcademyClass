window: 자바 스크립트 최상위 객체

  1. BOM(Browser Object Model): 브라우저의 주소창과 관련
  2. DOB(Document Object Model): BOM 제외 나머지 (html 요소 등)

window.open(): 브라우저의 새로운 탭(창)을 생성

About Timer

  1. setTimeout: 설정한 시간이 지난 후 로직 실행됨(비동기)
  2. setInterval: 설정한 시간마다 로직이 실행됨

BOM(Browser Object Model)

  1. location 객체
  2. screen 객체
  3. navigator 객체
  4. history 객체
>span< (클릭시 페이지이동)

DOB(Document Object Model)

노드(Node): HTML에 있는 각각의 태그
  - 요소(Element) 노드: 태그 그 자체를 의미
  - 텍스트(Text) 노드: 태그 내부의 텍스트를 의미

  - 텍스트(Text) 노드가 존재하는 요소: 시작, 종료 태그 사이에 문자열을 기입할 수 있는 태그
    (p, div, a, p, pre, span, h1~ ...)
  - 텍스트(Text) 노드가 존재하지 않는 요소: 시작, 종료 태그 사이에 문자열을 기입하지 않는 태그
    (img, input, br, hr ...)
