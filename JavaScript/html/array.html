배열 관련 내장함수

  • indexOf("값"): 값이 위치한 인덱스 반환, 없을경우 -1 반환
  • concat(배열, 배열, ...): 배열 합치기
  • reverse(): 배열의 값을 역순으로 재배치
  • sort(): 오름차순으로 정렬
  • push("값"): 배열의 맨 뒤에 요소 추가 -> 배열의 크기 반환
  • unshift("값"): 배열의 맨 앞에 요소 추가 -> 배열의 크기 반환
  • pop(): 배열의 맨 뒤 요소 삭제 -> 제거된 요소 반환
  • shift(): 배열의 맨 앞 요소 삭제 -> 제거된 요소 반환
  • slice(시작인덱스, 끝인덱스): 설정 인덱스에 위치한 요소 추출
  • join(구분자): 구분자를 기준으로 요소를 합치기
