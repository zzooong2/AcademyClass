HTMl은 웹 페이지를 만들기 위한 기본적인 마크업 언어입니다.

HTML 기본 구조

모든 HTML 문서는 다음과 같은 기본 구조를 가집니다. 

 
     < !DOCTYPE html >
     < html >
     < head >
     < title > 문서 제목  < /title >
     < /head >
     < body >
     
     < /body >
     < /html >
