Text

    font-size: 크기 변경 - px, %, ex, rm, vh ....

    text-align: 정렬

    • left: 왼쪽
    • center: 중심
    • right: 오른쪽

    font-weight: 굵기 변경

    • normal: 기본값
    • bold: 굵은 텍스트
    • 100 ~ 900: 값이 클수록 굵어짐

    font-style: 기울기 변경

    • normal: 기본값
    • oblique: 기울어진 텍스트
    • italic: 기울어진 텍스트

    text-decoration: 밑줄, 취소선, 윗줄 등의 효과

    • none: 기본값
    • underline: 밑줄
    • line-through: 취소선
    • overline: 윗줄
    • underline overline: 밑줄 + 윗줄

    text-shadow: 그림자 효과
