Layout

    block: 블럭 레벨 요소로 변경

                            - p 태그
                        - diplay: block;

    inline: 인라인 레벨 요소로 변경

                            - span 태그
                        - diplay: inline;

    inline-block: 컨텐츠 영역의 가로/세로 길이 변경 가능한 인라인 레벨 요소로 변경

                            - diplay: inline-block;

    none: 요소 숨김 (렌더링 O, 영역차지 X, 가시성 X)

                            - diplay: none;
