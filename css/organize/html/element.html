Element

    input type="text": 사용자로부터 값을 입력받을 때 사용

    input type="password": 사용자로부터 비밀번호 값을 입력받을 때 사용 -> 자동 마스킹

    input type:"number": 숫자를 입력받을 때 사용 (스핀박스 생성)

    input type="search, url, email": text 기능에 세분화된 기능을 제공 -> form 태그로 묶어줘야함




    textarea: 여러줄을 입력받을 때 사용

    input type="range": 슬라이드바를 사용한 숫자를 입력받음

    input type="datetime-local, date, month, week, time": 날짜, 시간을 입력 받을 때 사용

    input type="radio": 하나만 선택해야할 때 사용 - form태그 내부에 동일한 이름으로 선언된 요소만 선택 가능

    ⬆️⬆️⬆️⬆️ checkd 속성을 이용하면 항상 체크되어있음 ⬆️⬆️⬆️⬆️

    선택 안함 남자 여자

    input type="checkbox": 여러개 선택이 가능할 때 사용
